## Editing Workflow

### Use Case 1: Line Manager Changed

**Scenario:** An employee's line manager changes from Sarah Johnson to Robert Davis

**Steps:**
1. Navigate to the onboarding/exit request details page
2. Click **Edit Request** button
3. Update line manager fields:
   - Old: Sarah Johnson / sarah.johnson@company.com
   - New: Robert Davis / robert.davis@company.com
4. Click **Update Request**
5. Success message: "Request updated successfully"

### Use Case 2: Correcting Data Entry Error

**Scenario:** HR entered wrong email address for line manager

**Steps:**
1. Navigate to the request details page
2. Notice error in email (e.g., michael.browm@company.com instead of michael.brown@company.com)
3. Click **Edit Request** button
4. Correct the email address
5. Click **Update Request**
6. Verification: Email now displays correctly

### Use Case 3: Organizational Restructure

**Scenario:** Department merge results in new line manager for multiple employees

**Steps for each affected request:**
1. Open the request details page
2. Click **Edit Request** button
3. Update line manager information to reflect new structure
4. Add note explaining the change (optional)
5. Click **Update Request**
