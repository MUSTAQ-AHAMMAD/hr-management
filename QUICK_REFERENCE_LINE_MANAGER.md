# Quick Reference: Line Manager Feature Updates

## What Changed?

The line manager feature has been updated to allow HR to enter line manager information as **text fields** instead of selecting from a dropdown. This provides more flexibility and allows line managers to be updated at any time.

---

## Key Changes

### ✅ Before
- Line manager selected from dropdown (must be a system user)
- Email auto-filled and readonly
- Could not be edited after creation

### ✅ After  
- Line manager name and email entered as text fields
- Both fields are fully editable
- Can be updated anytime via Edit button

---

## Quick Start Guide

### Creating an Onboarding Request

1. Go to **Onboarding Requests** → **Create New Request**
2. Select the employee
3. **Enter line manager information** (optional):
   - Type the line manager's full name
   - Type the line manager's email address
4. Complete other fields and create the request

### Creating an Exit Clearance Request

1. Go to **Exit Clearance Requests** → **Create New Request**
2. Select the employee
3. **Enter line manager information** (required):
   - Type the line manager's full name (required)
   - Type the line manager's email address (required)
4. Complete other fields and create the request
5. Line manager will receive an approval email

### Editing Line Manager Information

1. Open the request details page
2. Click **Edit Request** button
3. Update line manager name and/or email
4. Click **Update Request** to save

---

## Important Notes

### For Onboarding Requests
- Line manager information is **optional**
- Can be left blank if not applicable
- Can be added or updated later

### For Exit Clearance Requests
- Line manager information is **required**
- An approval email is sent to the line manager
- Line manager must approve before HR can proceed
- Changing the email after approval does NOT resend the approval email

---

## Common Scenarios

### Scenario 1: Line Manager Changed
**Solution:** Edit the request and update the line manager name and email

### Scenario 2: Wrong Email Entered
**Solution:** Edit the request and correct the email address

### Scenario 3: Line Manager Not in System
**Solution:** No problem! Just type their name and email directly

### Scenario 4: Multiple Employees Under Same Manager
**Solution:** Enter the same line manager information for each request

---

## FAQ

**Q: Can I still use the old dropdown?**
A: No, the dropdown has been replaced with text fields for better flexibility.

**Q: What if the line manager is not a system user?**
A: That's fine! You can enter any name and email, even if they're not in the system.

**Q: Can I change the line manager after creating the request?**
A: Yes! Use the Edit button on the request details page.

**Q: Will the line manager receive notifications?**
A: For **exit clearance requests**, yes - they receive an approval email. For **onboarding requests**, notifications are optional.

**Q: What happens to existing requests?**
A: They continue to work as before. The system is backward compatible.

**Q: Can I leave line manager fields blank?**
A: For onboarding requests: yes (optional). For exit clearance requests: no (required).
